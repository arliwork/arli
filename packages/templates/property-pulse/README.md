# Property Pulse 🏠

**Real Estate Automation Platform**

Автоматизация для агентов недвижимости: скоринг лидов, рыночная аналитика, автоматический фоллоуап и управление сделками.

## Проблема

- Долгий цикл сделки (месяцы)
- 90% лидов — тайм-киллеры без намерений
- Ручной фоллоуап забывается
- CMA требует часов работы

## Решение

| Агент | Функция | Результат |
|-------|---------|-----------|
| Lead Scorer | Квалификация лидов по Intent, бюджету, таймлайну | Фокус на hot leads |
| Market Intel | Автоматические CMA, анализ neighborhood | Часы → минуты |
| Nurture Sequence | Персонализированный фоллоуап месяцами | Не упускаем лиды |
| Document Pilot | Сбор документов, трекинг дедлайнов | Никаких провалов |
| Listing Optimizer | AI-фото, описания, стратегия ценообразования | Быстрее продажи |

## Быстрый старт

```bash
arliwork init property-pulse --template ~/arli-templates/property-pulse
arliwork config --set mls.feed_url=https://
arliwork config --set crm.followup_boss.api_key=xxx
arliwork start
```
