# Hospitality Hive 🏨

**Restaurant & Hotel Operations**

Автоматизация ресторанов и отелей: оптимизация персонала, прогнозирование спроса, управление репутацией, yield management.

## Проблема

- Текучка персонала 100%+
- Непредсказуемый demand
- Негативные отзывы не отслеживаются
- Еда портится, теряем деньги

## Решение

| Агент | Функция | ROI |
|-------|---------|-----|
| Labor Optimizer | Прогноз busy periods, оптимальные schedule | -15% labor cost |
| Supply Chain | Предиктивный закуп, минимизация spoilage | -20% waste |
| Reputation Manager | Ответы на отзывы, actionable feedback | +рейтинг |
| Yield Manager | Dynamic pricing под occupancy | +10% revenue |
| Recipe Costing | Точный food cost per dish | +5% margin |

## Быстрый старт

```bash
arliwork init hospitality-hive --template ~/arli-templates/hospitality-hive
arliwork config --set pos.toast=true
arliwork config --set reservations.opentable=true
arliwork start
```

## Метрики

- Labor cost: -15%
- Occupancy rate: +10%
- Spoilage: -20%
- Response time на отзывы: <2 часов
