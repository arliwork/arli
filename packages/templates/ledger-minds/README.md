# Ledger Minds 📊

**Accounting & Advisory Automation**

Автоматизация бухгалтерии: категоризация транзакций, обнаружение аномалий, прогнозирование, отчётность клиентам и налоговая оптимизация.

## Проблема

- Месячный close занимает недели
- Ошибки в данных
- Реактивная работа вместо proactive advisory
- Tax planning — только в апреле

## Решение

| Агент | Функция | Результат |
|-------|---------|-----------|
| Transaction Categorizer | Auto-classify, learn from corrections | 85% автоматизации |
| Anomaly Hunter | Suspicious patterns, duplicates, fraud | Защита от потерь |
| Forecast Builder | Rolling forecasts, scenario modeling | Proactive advisory |
| Client Dashboard | Авто-отчёты без ручной работы | Довольные клиенты |
| Tax Optimizer | Year-round tax planning | Экономия клиентов |

## Быстрый старт

```bash
arliwork init ledger-minds --template ~/arli-templates/ledger-minds
arliwork config --set accounting.quickbooks=true
arliwork config --set payroll.gusto=true
arliwork start
```

## Метрики

- Точность категоризации: 85%+
- Скорость месячного close: -60%
- Время на advisory: +200%
- Налоговая экономия клиентов: +20%
