# Learning Foundry 🎓

**EdTech Content Automation**

Автоматизация образовательного контента: архитектура курсов, мультипликация контента, предсказание успеха студентов, генерация тестов.

## Проблема

- Создание курсов — месяцы
- Студенты бросают на 70%
- Нет персонализации
- Поддержка 24/7 невозможна

## Решение

| Агент | Функция | Влияние |
|-------|---------|---------|
| Curriculum Architect | Структурирование знаний в learning paths | Системный подход |
| Content Multiplier | Одна лекция → 5 форматов контента | 10x скорость |
| Student Success | Предиктивный поиск at-risk студентов | +30% completion |
| Assessment Generator | Персонализированные тесты | Лучше learning |
| Community Moderator | Модерация, ответы 24/7 | Engagement |

## Быстрый старт

```bash
arliwork init learning-foundry --template ~/arli-templates/learning-foundry
arliwork config --set lms.canvas=true
arliwork config --set community.discord=xxx
arliwork start
```

## Метрики

- Скорость создания курсов: 10x
- Completion rate: +30%
- Retention at-risk студентов: +40%
- Время ответа на вопросы: 24/7
