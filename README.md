A premium, dark-themed corporate website for ARLI — an AI research company building autonomous intelligence systems and multi-agent architectures.

## ✨ Features

### Design Philosophy
- **Minimalist Premium** — Clean, spacious layout like OpenAI/Anthropic
- **Dark Theme** — Deep blacks with subtle blue/purple accents
- **Glass Morphism** — Translucent cards with backdrop blur
- **Neural Network Visuals** — Animated node connections on canvas
- **Custom Cursor** — Interactive cursor with hover effects

### Sections
1. **Hero** — Bold headline with mission statement + neural network animation
2. **Mission Quote** — Manifesto block with gradient text
3. **Technology** — Multi-agent architecture explanation
4. **Research** — Academic papers showcase
5. **Agents** — 4 specialized AI agent types
6. **Careers** — Job listings + benefits
7. **Contact/CTA** — Waitlist signup form
8. **Footer** — Professional footer with links

### Premium Elements
- Animated particle network background
- GSAP scroll-triggered animations
- Counter animations for stats
- Aurora gradient effects
- Noise texture overlay
- Custom cursor (desktop)
- Glass morphism cards
- Smooth scroll navigation

## 📝 Content Customization

### Change Company Info
Edit in `index.html`:
- Company name: Search for "ARLI"
- Full name: "Automated Research & Linked Intelligence"
- Mission statement
- Research papers
- Job listings

### Update Colors
In the Tailwind config (inside `<script>`):
```javascript
colors: {
    arli: {
        500: '#0ea5e9',  // Primary
        // ... other shades
    }
}
```
