# Research Domain: Technology & Infrastructure

## Scope
Technical architecture, oracle solutions, blockchain infrastructure, and security for event derivatives platform.

## Key Hypotheses
1. Hybrid on-chain/off-chain architecture is optimal for institutional clients
2. UMA/Optimistic Oracle can settle 90%+ of events
3. Sub-100ms latency achievable for price updates
4. ICP (Internet Computer) provides HK-compliant decentralized compute
