# Research Domain: Regulatory & Compliance

## Scope
All legal and regulatory frameworks governing event contracts, prediction markets, and derivatives in target jurisdictions.

## Key Hypotheses
1. HK SFC can classify event contracts as derivatives under SFO (not gambling)
2. Singapore MAS has a viable licensing path via RMO or CMS
3. Dubai VARA provides the most crypto-friendly regulated framework
4. US CFTC path is viable but slow (Kalshi precedent)

## Data Points to Track
| Metric | Source | Update Freq |
|--------|--------|-------------|
| SFC consultation status | info.gov.hk | Daily |
| MAS regulatory updates | mas.gov.sg | Weekly |
| VARA guidance releases | var.ae | Weekly |
| CFTC no-action letters | cftc.gov | Weekly |
| Legislative bills (LegCo) | legco.gov.hk | Daily |

## Success Criteria
- Clear licensing path identified for each jurisdiction
- Regulatory risks quantified (fine amounts, enforcement history)
- Timeline to license documented
- Compliance costs estimated
